Our Mission

Drinks Catalog is a web application built with Flask and MongoDB that allows bars to manage their catalog of drinks. It provides a simple and intuitive interface to add, view, edit, and delete drinks from the catalog.

Our Security

The application features a user authentication system that allows bars to create an account and manage their own catalog of drinks. This makes it easy for bars to keep track of their inventory and update their drink offerings as needed.
